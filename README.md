# SPARCS Taxi
KAIST Taxi Party Matching Web Service

## About
Taxi는 KAIST 구성원들의 택시 동승 인원 모집을 위한 서비스입니다. 🚖
비교적 거리가 먼 장소에 갈 때 택시 동승이 빈번하게 발생하는 점을 인지하고, 이를 어플/웹 서비스를 통해 보다 편리하게 이루어지도록 하고자 합니다.
